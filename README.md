# 📋 Internship Report — Natdanai Phothong CED13
เว็บไซต์รายงานผลการฝึกปฎิบัติงานสหกิจศึกษา (Vite Project)

## 🚀 เริ่มใช้งาน

```bash
# 1. เข้าโฟลเดอร์
cd intern-report-vite

# 2. ติดตั้ง dependencies
npm install

# 3. รัน dev server
npm run dev
```

เปิดเบราว์เซอร์ที่ → http://localhost:5173

## 📦 Build สำหรับ Production

```bash
npm run build
npm run preview
```

## 📁 โครงสร้างไฟล์

```
intern-report-vite/
├── index.html          ← หน้าแรก
├── workplace.html      ← สถานที่ฝึกงาน

├── about.html          ← ข้อมูลผู้จัดทำ
├── vite.config.js      ← Vite MPA config
├── src/
│   ├── main.js         ← JavaScript entry

└── package.json
```
